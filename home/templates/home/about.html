The Components of Face Recognition

Here's a quick blurb about the purpose of the image understanding lab.

What Makes Face Recognition so Difficult?

  1. Perceptually similar
  2. Must recognize faces across viewpoint, scale, and lighting conditions
  3. Requires a link from representation of face to memory of identity

Why This Test?

Using the same match-to-sample paradigm, we can assess your competency with respect to each of these dimensions:

  1. Perceptual similarity
  2. Memory
  3. Viewpoint invariance

Consider memory and viewpoint independently from perceptual similarity link to Gabor similarity web app.
